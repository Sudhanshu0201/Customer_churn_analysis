# 📊 Customer Churn Analysis

> An end-to-end Data Analytics project that analyzes customer churn using Python, PostgreSQL, and Power BI to generate business insights and support data-driven decision-making.

---

## 📌 Overview

Customer churn is one of the biggest challenges for subscription-based businesses. This project analyzes customer behavior to identify the key factors influencing churn and provides actionable insights to improve customer retention.

The project follows a complete analytics workflow, starting from raw data processing and exploratory analysis to SQL-based business querying, interactive dashboard creation, and business reporting.

---

## 🎯 Business Objectives

- Analyze customer demographics and subscription behavior.
- Identify factors contributing to customer churn.
- Perform data cleaning and exploratory data analysis (EDA).
- Generate business insights using PostgreSQL.
- Develop an interactive Power BI dashboard.
- Present findings through a business report and presentation.

---

## 📂 Dataset

The dataset contains customer information, including:

- Customer Demographics
- Subscription Details
- Contract Type
- Payment Method
- Internet & Service Usage
- Customer Tenure
- Monthly & Total Charges
- Churn Status

**Target Variable:** `Churn`

# 🔄 Project Workflow

### 1️⃣ Data Loading
- Imported the raw dataset using Pandas.
- Explored dataset structure and feature information.

### 2️⃣ Data Cleaning
- Handled missing values.
- Removed duplicate records.
- Corrected inconsistent data types.
- Created new analytical features.
- Prepared data for analysis.

### 3️⃣ Exploratory Data Analysis (EDA)
Performed detailed analysis to understand customer behavior by analyzing:

- Churn Distribution
- Customer Demographics
- Contract Type
- Payment Method
- Internet Service
- Customer Tenure
- Monthly Charges
- Total Revenue

Created multiple visualizations to identify trends, patterns, and relationships affecting churn.

### 4️⃣ SQL Analysis (PostgreSQL)
Loaded the cleaned dataset into PostgreSQL and answered business questions using SQL.

Examples include:

- Overall Churn Rate
- Revenue by Customer Segment
- Churn by Contract Type
- Churn by Payment Method
- Customer Distribution
- Service-wise Analysis
- Revenue Metrics

### 5️⃣ Power BI Dashboard
Built an interactive dashboard with:

- Executive KPI Cards
- Churn Rate Analysis
- Customer Segmentation
- Revenue Analysis
- Contract Analysis
- Payment Method Analysis
- Service Analysis
- Interactive Filters & Slicers

### 6️⃣ Business Report
Prepared a business report summarizing:

- Key Findings
- Business Insights
- Churn Drivers
- Recommendations

### 7️⃣ Business Presentation
Created a presentation to communicate insights, conclusions, and recommendations to stakeholders.

# 🚀 How to Run

### Clone the Repository

```bash
git clone https://github.com/Sudhanshu0201/Customer_churn_analysis.git
```

### Navigate to the Project

```bash
cd Customer_churn_analysis
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Project

1. Open the Jupyter Notebook.
2. Execute the notebooks in sequence.
3. Generate the cleaned dataset.
4. Import the cleaned data into PostgreSQL.
5. Execute the SQL queries.
6. Open the Power BI (.pbix) file.
7. Refresh the data source.
8. Explore the interactive dashboard.
